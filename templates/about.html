About This Application

Purpose

This web application uses machine learning to assist in breast cancer diagnosis by analyzing cell nuclei characteristics from fine needle aspirate (FNA) samples. The model predicts whether a tumor is benign or malignant based on various measurements.

How It Works

Advanced Model Selection

Trains 7 different algorithms and automatically selects the best performing one based on F1 score

Data Cleaning

Uses Z-score outlier detection to remove anomalous data points for better accuracy

30 Features

Analyzes 30 different measurements of cell nuclei characteristics

Validated Performance

Models tested on separate validation and test sets for reliable performance metrics

Model Training Process

Our system follows a rigorous machine learning pipeline:

  1. Data Cleaning: Removes outliers using Z-score analysis (|Z| > 3)
  2. Feature Scaling: Standardizes all 30 features using StandardScaler
  3. Model Training: Trains 7 different algorithms simultaneously
  4. Validation: Evaluates each model on separate validation data
  5. Selection: Automatically chooses the best performing model
  6. Testing: Final evaluation on unseen test data
